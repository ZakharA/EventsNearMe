## Project description
A ASP.Net website where users can browse and search events around them. Written using ASP.NET MVC, JS, CSS, HTML. 

Main features:
 * Registration
 * Login
 * CRUD events
 * User roles (Admin, Event Organizer, User) 
 
## Technologies
Project is created with:
 * ASP.NET MVC
 * Javascript
 * CSS
 * HTML
